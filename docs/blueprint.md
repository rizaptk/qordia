# **App Name**: Qordia: Scan, Order, Serve

## Core Features:

- QR Code Scanning: Scan QR codes at tables to identify seating location and outlet.
- Digital Menu: Browse categorized menus with images, descriptions, and real-time availability.
- Order Customization: Customize orders with options for sugar level, ice level, toppings, and special notes.
- Order Tracking: Monitor order progress in real-time with status updates.
- Preparation Display System: Display incoming orders with item breakdown, customization notes, and table reference for staff.
- Sales and Performance Analytics: Provide insights on best-selling items, peak hours, table utilization, and sales performance.
- Suggested Item Recommendations: Suggest popular and relevant menu items based on current order and historical data.

## Style Guidelines:

- Primary color: Royal Smart Purple (#6B5CFF) for main brand identity and key interface components.
- Secondary Accent Color: Fresh Order Mint (#2ED1A2) for success states and order confirmation.
- Supporting Hospitality Tone: Soft Amber (#FFB547) for promotions and menu highlights.
- Neutral Interface Colors: Dark Text/Dashboard Primary (#1F2937), Light Background (#F8FAFC), Border and Divider (#E5E7EB).
- Primary Font: Plus Jakarta Sans for a friendly and modern feel with excellent readability.
- Minimal geometric design with rounded corner shapes and balanced whitespace.
- Clean and intuitive layout prioritizing operational efficiency and simplicity, utilizing daisyUI's grid and flexbox components.
- Subtle transitions and animations providing smooth user feedback without decorative complexity, employing daisyUI's transition classes for hover and focus states.
